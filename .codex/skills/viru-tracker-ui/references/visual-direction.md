# Visual Direction — Viru Tracker

- Mantener jerarquia fuerte: titulo > contenido > acciones.
- Preservar composicion editorial con ritmo y densidad util.
- Usar paneles de borde fino, radios consistentes y sombras suaves.
- No aplanar la UI hacia grids genericos de cards iguales.
- Reforzar estados vacios con recursos sobrios (ghost grid/textura tenue), no ruido visual.
- Respetar paleta y tokens canonicos; no introducir colores fuera de base.
- Semantica de estado unificada: `success`, `warning`, `error`, `info`.
- Microinteracciones cortas: feedback tactil, foco visible, transiciones discretas.
- Priorizar mejoras pequenas, legibles y comprobables por ruta.
